- **Status-based colors**:

  - Yellow (16-30 days): `warning-500`
  - Orange (31-60 days): `#F97316` (custom orange)
  - Red (61-90 days): `error-500`
  - Black (91+ days): `charcoal-900`
- **Immigration alerts**: Red badges for expiring visas

### Key Components
- **Utilization bar**: Prominent, always visible
- **Consultant cards**: Photo, name, days on bench, visa status
- **Status tabs**: Quick filter by bench urgency
- **Hotlist builder**: Drag-and-drop selection
- **Immigration dashboard**: Timeline view of expiring visas

### Forms
- **Consultant onboard**: Wizard (Profile â†’ Skills â†’ Work Auth â†’ Documents)
- **Vendor form**: Single page with sections
- **Hotlist**: Multi-select grid with preview

### Unique Patterns
- **Color-coded everything**: Bench days drive visual priority
- **Countdown timers**: Visa expiration prominently shown
- **Quick actions**: Submit to job, add to hotlist, update status
