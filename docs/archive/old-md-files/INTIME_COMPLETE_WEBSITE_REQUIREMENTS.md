## PART XIII: CONTENT GUIDELINES

### Voice and Tone

**Professional but Approachable:**
- Use "we" and "you" for connection
- Avoid jargon unless necessary
- Explain complex concepts simply
- Maintain authority without arrogance

**Action-Oriented:**
- Start with verbs in CTAs
- Focus on outcomes, not features
- Use active voice predominantly
- Create urgency without pressure

**Transformation-Focused:**
- Emphasize change and growth
- Use before/after contrasts
- Highlight journey, not destination
- Celebrate milestones and achievements

### Content Patterns

**Headlines:**
- Lead with benefit or outcome
- Use power words (Transform, Power, Excel)
- Create curiosity or promise value
- Keep under 10 words when possible

**Subheadlines:**
- Expand on headline promise
- Add specificity and context
- Include proof points if relevant
- Bridge to body content

**Body Copy:**
- Lead with most important info
- Use short paragraphs (3-4 lines max)
- Include bullet points for scanning
- End with clear next step

**CTAs:**
- Use action verbs (Start, Get, Transform)
- Be specific about outcome
- Create urgency when appropriate
- Provide alternative options

### SEO Considerations

**Page Titles:**
- Include primary keyword
- Keep under 60 characters
- Include brand name at end
- Make compelling for clicks

**Meta Descriptions:**
- Summarize page value
- Include call-to-action
- Keep under 155 characters
- Use target keywords naturally

**Header Structure:**
- One H1 per page
- Logical H2-H3 hierarchy
- Include keywords in headers
- Make scannable and clear

**Image Optimization:**
- Descriptive file names
- Alt text for accessibility
- Compressed for performance
- Relevant to content
