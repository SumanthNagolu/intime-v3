# 1. Introduction

## 1.1 About This Manual

Welcome to the InTime eSolutions HR User Manual. This comprehensive guide is designed to help Human Resources professionals navigate and utilize the full capabilities of the InTime HR Management System. Whether you're managing a small team or overseeing an entire organization's HR operations, this manual will provide step-by-step instructions for every feature.

## 1.2 Who Should Use This Manual

This manual is intended for:

- **HR Managers** - Full system access for all HR operations
- **HR Coordinators** - Day-to-day HR tasks and employee management
- **Recruiters** - Employee onboarding and lifecycle management
- **Department Managers** - Team management, approvals, and performance reviews
- **Finance/Payroll Administrators** - Payroll processing and compensation management
- **System Administrators** - Configuration and maintenance

## 1.3 System Requirements

To access InTime eSolutions, you need:

- **Web Browser**: Chrome 90+, Firefox 85+, Safari 14+, or Edge 90+
- **Internet Connection**: Stable broadband connection
- **Screen Resolution**: Minimum 1280x720 (1920x1080 recommended)
- **Authentication**: Valid organizational email and password

## 1.4 Support Contact Information

For technical support or questions:

- **Email**: support@intimeesolutions.com
- **Phone**: 1-800-INTIME-HR (1-800-468-4634)
- **Help Desk**: Available Monday-Friday, 9 AM - 6 PM EST

# 2. Getting Started

## 2.1 Logging Into the System

### Step-by-Step Login Process

1. **Open your web browser** and navigate to your organization's InTime portal URL
   - Example: `https://yourcompany.intimeesolutions.com`

2. **Enter your credentials**:
   - Email address (your organizational email)
   - Password

3. **Click the "Sign In" button**

4. **Multi-Factor Authentication** (if enabled):
   - Enter the verification code sent to your phone or authenticator app
   - Click "Verify"

5. **Dashboard**: Upon successful login, you'll see your personalized dashboard

### First-Time Login

If this is your first time logging in:

1. Click "Forgot Password" or "First Time User"
2. Enter your organizational email
3. Check your email for a password setup link
4. Create a strong password (minimum 8 characters, including uppercase, lowercase, number, and special character)
5. Log in with your new credentials

# 4. Module 1: People Management

## 4.1 Employees

### Overview

The Employees section is the central hub for managing all employee records in your organization. This includes both full-time employees and contractors.

**Navigation**: Operations â†’ People â†’ Employees

### 4.1.1 Viewing the Employee List

When you access the Employees section, you'll see:

**Statistics Dashboard** (Top of page):
- **Total Employees**: Count of all employee records
- **Active**: Currently active employees
- **New This Month**: Employees added in current month
- **Departments**: Total number of departments

**Search and Filters**:
- Search bar: Find employees by name, title, or department
- Status filter: All Status, Onboarding, Active, On Leave, Terminated
- Employment Type filter: All Types, FTE, Contractor, Intern, Part-Time

**Employee Table Columns**:
| Column | Description |
|--------|-------------|
| Name | Employee full name with avatar |
| Title | Job title/position |
| Department | Assigned department |
| Status | Employment status badge |
| Email | Work email address |
| Phone | Contact phone number |
| Manager | Direct supervisor name |
| Employment Type | FTE/Contractor/Intern/Part-Time |
| Hire Date | Date of employment start |
| Location | Work location |

### 4.1.2 Adding a New Employee

**Step-by-Step Process**:

1. **Click "+ ADD EMPLOYEE"** button (top right corner)

2. **Basic Information Tab**:

   | Field | Description | Required |
   |-------|-------------|----------|
   | First Name | Employee's first name | Yes |
   | Last Name | Employee's last name | Yes |
   | Email | Work email address | Yes |
   | Phone | Contact phone number | No |
   | Employee Number | Unique ID (auto-generated if blank) | No |

3. **Employment Details Tab**:

   | Field | Description | Options |
   |-------|-------------|---------|
   | Status | Employment status | Onboarding, Active |
   | Employment Type | Type of employment | FTE, Contractor, Intern, Part-Time |
   | Hire Date | Start date of employment | Date picker |
   | Department | Assigned department | Dropdown list |
   | Job Title | Position title | Text input |
   | Manager | Direct supervisor | Employee dropdown |

4. **Compensation Tab** (Optional):

   | Field | Description |
   |-------|-------------|
   | Salary Type | Annual, Hourly, Monthly |
   | Salary Amount | Compensation amount |
   | Currency | USD, EUR, GBP, etc. |

5. **Work Details Tab**:

   | Field | Description | Options |
   |-------|-------------|---------|
   | Location | Work location | Office locations dropdown |
   | Work Mode | Working arrangement | On-site, Remote, Hybrid |

6. **Review and Save**:
   - Review all entered information
   - Click **"Save Employee"** to create the record
   - System will display success message
   - New employee appears in the list

### 4.1.3 Editing an Employee Record

1. **Find the employee** using search or filters
2. **Click on the employee row** to open the detail view
3. **Click "Edit"** button in the employee header
4. **Modify the necessary fields**
5. **Click "Save Changes"** to update the record

### 4.1.4 Employee Status Management

**Status Definitions**:

| Status | Description | Typical Use |
|--------|-------------|-------------|
| **Onboarding** | New hire going through orientation | First 30-90 days |
| **Active** | Currently employed and working | Standard status |
| **On Leave** | Temporary leave of absence | LOA, parental leave |
| **Terminated** | Employment has ended | Resignation, termination |

**Changing Employee Status**:

1. Open the employee detail view
2. Click the status badge dropdown
3. Select the new status
4. If terminating:
   - Enter termination date
   - Select termination type (Voluntary/Involuntary/Retirement/Other)
   - Add termination notes (optional)
5. Confirm the status change

### 4.1.5 Filtering and Searching Employees

**Using the Search Bar**:
- Type employee name, job title, or department
- Results filter automatically as you type
- Press Enter or wait for auto-refresh

**Using Filters**:

1. **Status Filter**:
   - Click "All Status" dropdown
   - Select: Onboarding, Active, On Leave, or Terminated
   - List updates immediately

2. **Employment Type Filter**:
   - Click "All Employment Types" dropdown
   - Select: FTE, Contractor, Intern, or Part-Time

3. **Combining Filters**:
   - Both filters work together
   - Example: Active + FTE shows only active full-time employees

**Sorting the Table**:
- Click any column header to sort
- Click again to reverse sort order
- Arrow indicator shows current sort direction

### 4.1.6 Exporting Employee Data

1. Click the **"Export"** button (if available)
2. Select export format: CSV, Excel, or PDF
3. Choose which fields to include
4. Click **"Download"**

**Step-by-Step Process**:

1. **Click "+ ADD DEPARTMENT"** button

2. **Fill in Department Details**:

   | Field | Description | Required |

   | Department Head | Manager/leader of department | No |
   | Cost Center Code | Financial tracking code | No |
   | Budget Amount | Department budget | No |
   | Status | Active, Inactive, or Archived | Yes |

3. **Click "Save Department"**

# 11. Troubleshooting & FAQ

## 11.1 Common Issues

### Issue: Cannot find an employee

**Possible Causes & Solutions**:
1. **Filters are applied** â†’ Clear all filters and search again
2. **Wrong status selected** â†’ Check "All Status" filter
3. **Employee in different department** â†’ Remove department filter
4. **Spelling error** â†’ Try partial name search
5. **Employee terminated** â†’ Check "Terminated" status filter

### Issue: Timesheet not showing for approval

**Possible Causes & Solutions**:
1. **Not submitted** â†’ Confirm employee clicked "Submit"
2. **Wrong approver** â†’ Check manager assignment
3. **Different pay period** â†’ Adjust date filter
4. **Already approved** â†’ Check "Approved" status

### Issue: Leave balance incorrect

**Possible Causes & Solutions**:
1. **Pending requests** â†’ Check for approved but unused leave
2. **Accrual not processed** â†’ Contact HR to verify accrual
3. **Recent hire** â†’ Pro-rated balance may apply
4. **Policy change** â†’ New policy may have different allocation

### Issue: Cannot start onboarding for new hire

**Possible Causes & Solutions**:
1. **Employee not created** â†’ Create employee record first
2. **Status not "Onboarding"** â†’ Change status to Onboarding
3. **Onboarding already started** â†’ Check existing onboardings

### Issue: Performance review not visible

**Possible Causes & Solutions**:
1. **Cycle not started** â†’ Verify cycle is in active phase
2. **Employee excluded** â†’ Check cycle participants
3. **Wrong phase** â†’ Confirm you're in correct review phase
4. **Hire date filter** â†’ New employees may be excluded

## 11.2 Frequently Asked Questions

**Q: How do I change an employee's department?**
A: Edit the employee record, change the Department field, and save. The org chart will update automatically.

**Q: Can I delete a pay run that's already processed?**
A: No, processed pay runs cannot be deleted. You can void the pay run and create a correcting entry.

**Q: How far in advance can employees request leave?**
A: There's no system limit. Your company policy should define any restrictions.

**Q: Can managers see all employee salaries?**
A: Only if they have compensation access. Standard managers see only their direct reports (if configured).

**Q: How do I run a report for terminated employees?**
A: Filter the Employees list by "Terminated" status, then use the export function.

**Q: Can I customize the onboarding checklist for different roles?**
A: Yes, create role-specific templates in the Onboarding Templates section.

**Q: What happens to an employee's data when they're terminated?**
A: Data is retained for compliance. The record is marked as terminated but not deleted.

**Q: How do I add a new leave type?**
A: Contact your system administrator. Leave policies require admin-level configuration.
