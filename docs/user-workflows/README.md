# User Workflows Documentation

This folder contains detailed workflows for each role in the InTime staffing platform.

## Contents

- **END_USER_WORKFLOWS.md** - Comprehensive workflows organized by role:
  - Recruiter workflows
  - Account Manager workflows
  - Sales workflows
  - Operations workflows
  - Manager workflows
  - Executive workflows
  - System Admin workflows

## Workflow Format

Each workflow follows a consistent structure:

1. **What Happens Today** - Current manual process and pain points
2. **What Should Happen** - Ideal AI-assisted workflow
3. **Success Criteria** - How to measure improvement

## Purpose

These workflows serve as:
- Requirements documentation for feature development
- User story source for sprint planning
- Training material for new team members
- Baseline for measuring AI efficiency gains
