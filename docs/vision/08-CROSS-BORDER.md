## H-1B Process (USA)

### Overview

**H-1B** = US work visa for specialized occupations (including Guidewire developers)

**Key Differences vs LMIA:**
- **Lottery system:** Limited H-1Bs per year (85,000 cap), must win lottery
- **Timing:** Can only apply in March/April (once per year)
- **Employer-sponsored:** US employer must sponsor (like LMIA)
- **Timeline:** 6-12 months (longer than Canada)

### H-1B Process (High-Level)

```
MARCH: H-1B Registration ($10 fee per registration)
â”œâ”€ Employer registers candidate in lottery
â”œâ”€ Lottery results: April (20-30% chance of selection)
â””â”€ If selected: Proceed to full application

APRIL-JUNE: H-1B Petition (if lottery selected)
â”œâ”€ File Form I-129 (H-1B petition)
â”œâ”€ Labor Condition Application (LCA) - prove wage is market rate
â”œâ”€ Supporting docs: Degree, resume, job offer
â”œâ”€ Processing: 3-6 months (or 15 days with premium processing +$2,500)
â””â”€ Approval: I-797 (H-1B approval notice)

OCTOBER: H-1B Start Date
â”œâ”€ Earliest start date: October 1st (fiscal year)
â”œâ”€ Candidate can begin working (if in US)
â”œâ”€ If outside US: Apply for H-1B visa stamp at US consulate
â””â”€ Enter US, start work

TOTAL TIMELINE: March (lottery) â†’ October (start) = 7 months minimum
```
