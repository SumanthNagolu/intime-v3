# Use Case: Track Commission

## Overview

| Property | Value |
|----------|-------|
| Use Case ID | UC-REC-G05 |
| Actor | Recruiter |
| Goal | Monitor commission earnings from placements using 5% of gross share model |
| Frequency | Weekly/Monthly review |
| Estimated Time | 5-10 minutes |
| Priority | High |

---

## Preconditions

1. User is logged in as Recruiter
2. Active placements exist
3. Timesheet data available
4. Commission structure defined (5% of gross)

---

## Trigger

- Monthly commission review
- Placement starts (commission activates)
- Payroll processing period
- Quarterly earnings review
- Dispute or question about commission

## Commission Model

### 5% of Gross Share

```
Commission = Gross Billing Ã— 5%

Where:
  Gross Billing = Bill Rate Ã— Hours Worked

Example:
  Bill Rate: $125/hr
  Hours: 160/month
  Gross Billing: $20,000
  Commission: $20,000 Ã— 5% = $1,000/month
```

| Event | Commission Status |

| Contractor starts work | Accruing |
| Timesheet approved | Calculated |
| Invoice paid | Approved for payout |
| Payout date | Paid |
