## 1. Workflow Overview

### Purpose

Enable candidates to discover relevant job opportunities and submit applications directly through the Candidate Portal.

### Scope

- Search and filter jobs by location, skills, job type
- View detailed job descriptions
- Apply with one-click or custom application
- Track application status

### Preconditions

- Candidate has registered and verified email
- Profile is at least 60% complete (for applications)
- Candidate is not currently on an active assignment

### Success Criteria

- Candidate finds relevant jobs within 3 clicks
- Application submission < 60 seconds for returning candidates
- 95%+ application success rate (no technical errors)

## 2. User Stories

### US-CP-001: Search Jobs by Skill
**As a** candidate
**I want to** search for jobs matching my skills
**So that** I can find relevant opportunities

**Acceptance Criteria:**
- Search box supports keyword search
- Skills from my profile auto-suggest
- Results ranked by relevance score
- Show match percentage for each job

### US-CP-002: Filter Jobs by Location
**As a** candidate
**I want to** filter jobs by location and remote options
**So that** I can find jobs I can commute to or work remotely

**Acceptance Criteria:**
- Filter by city, state, country
- Filter by remote/hybrid/onsite
- "Willing to relocate" option
- Show distance from my location

### US-CP-003: Quick Apply
**As a** candidate with a complete profile
**I want to** apply with one click
**So that** I can submit applications quickly

**Acceptance Criteria:**
- "Quick Apply" button visible if profile >= 80%
- No additional forms required
- Confirmation within 2 seconds
- Option to add cover letter (optional)

### US-CP-004: Save Jobs for Later
**As a** candidate
**I want to** save jobs I'm interested in
**So that** I can apply when ready

**Acceptance Criteria:**
- Save/bookmark button on each job card
- "Saved Jobs" section in dashboard
- Notification if saved job is about to expire
- Maximum 50 saved jobs

## 9. Business Rules

### BR-CP-001: Profile Completeness for Application
```
IF candidate.profile_completeness < 60% THEN
  - Block application submission
  - Show "Complete your profile to apply"
  - Link to profile completion wizard
END IF
```

### BR-CP-002: Quick Apply Eligibility
```
IF candidate.profile_completeness >= 80%
   AND candidate.resume_uploaded = true
   AND candidate.work_authorization_verified = true THEN
  - Show "Quick Apply" button
  - Allow one-click submission
ELSE
  - Show "Apply" button (full form)
END IF
```

### BR-CP-003: Duplicate Application Prevention
```
IF candidate.applications.exists(job_id)
   AND application.status NOT IN ('withdrawn', 'rejected') THEN
  - Block new application
  - Show "You've already applied to this job"
  - Link to existing application
END IF
```

### BR-CP-004: Application Rate Validation
```
IF expected_rate < job.rate_min * 0.85 THEN
  - Show warning: "Your rate is below the job range"
  - Allow submission anyway
END IF

IF expected_rate > job.rate_max * 1.15 THEN
  - Block submission
  - Show error: "Expected rate exceeds job maximum"
END IF
```

### BR-CP-005: Job Visibility Rules
```
Jobs visible to candidates must match ALL:
- job.status = 'published'
- job.is_public = true
- job.expires_at > NOW() OR job.expires_at IS NULL
- job.positions_filled < job.positions_open
```

### BR-CP-006: Match Score Calculation
```
match_score = (
  skills_match_weight * skills_match_percent +      // 40%
  experience_match_weight * experience_match +       // 25%
  location_match_weight * location_match +           // 20%
  availability_match_weight * availability_match     // 15%
) / 100
```

## 11. Error Scenarios

### ES-001: Profile Incomplete
**Trigger:** Candidate attempts to apply with < 60% profile
**Message:** "Please complete your profile before applying. You need at least 60% completeness."
**Action:** Link to profile wizard

### ES-002: Already Applied
**Trigger:** Candidate applies to same job twice
**Message:** "You've already applied to this position on [date]."
**Action:** Link to existing application, offer to update

### ES-003: Job No Longer Available
**Trigger:** Job closed/filled between search and apply
**Message:** "This position is no longer accepting applications."
**Action:** Show similar jobs, remove from saved

### ES-004: Resume Upload Failed
**Trigger:** Resume file too large or wrong format
**Message:** "Resume must be PDF, DOC, or DOCX and under 5MB."
**Action:** Show file selector again

### ES-005: Rate Out of Range
**Trigger:** Expected rate exceeds job max by > 15%
**Message:** "Your expected rate of $X/hr exceeds the job's maximum of $Y/hr."
**Action:** Allow edit, suggest within-range rate

### ES-006: Network Error
**Trigger:** Connection lost during application
**Message:** "Connection lost. Your progress has been saved."
**Action:** Auto-save draft, retry button

### ES-007: Session Expired
**Trigger:** Session timeout during application
**Message:** "Your session has expired. Please log in again."
**Action:** Save draft, redirect to login, return to application
