# First 90 Days Guide

Your roadmap to success in your first three months at InTime.

---

## Overview

| Phase | Timeframe | Focus |
|-------|-----------|-------|
| Onboarding | Days 1-7 | Learn the basics |
| Learning | Days 8-30 | Understand the business |
| Building | Days 31-60 | Start contributing |
| Performing | Days 61-90 | Work independently |

---

## Days 1-7: Onboarding Phase

### Day 1: Welcome

**Morning:**
- [ ] Complete HR paperwork
- [ ] Set up email and Slack
- [ ] Get system access
- [ ] Meet your team lead

**Afternoon:**
- [ ] Read [Company Bible](../01-foundation/COMPANY_BIBLE.md)
- [ ] Tour workspace (virtual or physical)
- [ ] Meet immediate team members
- [ ] Review your role handbook

**End of Day:**
- [ ] Check-in with manager
- [ ] Confirm Day 2 schedule

### Days 2-3: Foundation

- [ ] Complete [US IT Staffing 101](../06-training/guides/US_IT_STAFFING_101.md) (Chapter 1-3)
- [ ] Watch training videos (if available)
- [ ] Shadow team members
- [ ] Learn key processes
- [ ] Set up all tools

### Days 4-5: Immersion

- [ ] Complete US IT Staffing 101 (Chapters 4-6)
- [ ] Attend all team meetings
- [ ] Practice with mock scenarios
- [ ] Review past work examples
- [ ] Ask lots of questions

### Weekend: Reflection

- What did I learn this week?
- What do I still not understand?
- What questions do I need to ask?

---

## Days 8-30: Learning Phase

### Week 2: Deep Dive

**Goals:**
- Understand your role deeply
- Learn all relevant systems
- Build relationships with team
- Complete training requirements

**Activities:**
- [ ] Complete role-specific handbook
- [ ] Observe 5+ real work scenarios
- [ ] Practice tasks with supervision
- [ ] Daily check-in with manager
- [ ] Meet cross-functional team members

**By End of Week 2:**
- Can explain what InTime does
- Know all team member roles
- Understand basic processes
- Comfortable with tools

### Week 3: Practice

**Goals:**
- Handle tasks with supervision
- Apply what you've learned
- Get feedback and improve
- Build confidence

**Activities:**
- [ ] Complete first tasks independently
- [ ] Get feedback on every task
- [ ] Attend 1:1 with manager
- [ ] Shadow more complex scenarios
- [ ] Document questions and learnings

**By End of Week 3:**
- Completed 3-5 tasks independently
- Received constructive feedback
- Identified areas for improvement
- Building confidence

### Week 4: Assessment

**Goals:**
- Demonstrate competency
- 30-day review with manager
- Set goals for next 30 days
- Identify development areas

**Activities:**
- [ ] Complete 30-day self-assessment
- [ ] 30-day review meeting with manager
- [ ] Set specific goals for Days 31-60
- [ ] Identify training needs
- [ ] Celebrate wins!

**30-Day Review Topics:**
```
1. What have you learned?
2. What's going well?
3. What's challenging?
4. How can we support you?
5. Goals for next 30 days
```

---

## Days 31-60: Building Phase

### Week 5-6: Increasing Responsibility

**Goals:**
- Handle more tasks independently
- Reduce supervision needed
- Build speed and accuracy
- Take ownership of outcomes

**Activities:**
- [ ] Own assigned work fully
- [ ] Minimal supervision needed
- [ ] Ask for more responsibility
- [ ] Help newer team members
- [ ] Contribute to team discussions

**By End of Week 6:**
- 80% of tasks done independently
- Quality meets expectations
- Building speed
- Known as reliable

### Week 7-8: Contribution

**Goals:**
- Full workload at expected pace
- Proactive problem-solving
- Positive impact on team
- Identify improvements

**Activities:**
- [ ] Carry full workload
- [ ] Solve problems before escalating
- [ ] Suggest process improvements
- [ ] Build cross-functional relationships
- [ ] Prepare for 60-day review

**60-Day Review Topics:**
```
1. Progress on 30-day goals
2. Areas of strength
3. Areas for development
4. Feedback on role/team
5. Goals for Days 61-90
```

---

## Days 61-90: Performing Phase

### Week 9-10: Independence

**Goals:**
- Work fully independently
- Meet or exceed metrics
- Contribute to team success
- Develop expertise

**Activities:**
- [ ] Consistent performance
- [ ] No supervision needed
- [ ] Mentor newer team members
- [ ] Take on stretch assignments
- [ ] Build reputation for quality

**By End of Week 10:**
- Performing at expected level
- Trusted to work independently
- Contributing beyond basic role
- Building expertise

### Week 11-12: Ownership

**Goals:**
- Own outcomes completely
- Exceed expectations
- Lead small initiatives
- Prepare for 90-day review

**Activities:**
- [ ] Exceed performance targets
- [ ] Lead improvement project
- [ ] Share knowledge with team
- [ ] Build external relationships
- [ ] Prepare 90-day review

**90-Day Review Topics:**
```
1. Progress and achievements
2. Performance vs. expectations
3. Growth demonstrated
4. Goals for next quarter
5. Career development plan
```

## Common Challenges

### Feeling Overwhelmed

**It's normal to feel like there's too much to learn.**

Solutions:
- Focus on one thing at a time
- Write everything down
- Ask questions early
- Take breaks when needed
- Celebrate small wins

### Making Mistakes

**Everyone makes mistakes when learning.**

Solutions:
- Own it quickly
- Learn from it
- Ask how to prevent it
- Don't repeat it
- Move forward

### Imposter Syndrome

**Feeling like you don't belong is common.**

Solutions:
- Remember you were hired for a reason
- Compare to yesterday's you, not others
- Celebrate progress
- Ask for feedback
- Trust the process
