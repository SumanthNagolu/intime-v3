## Email Setup (Google Workspace)

### Initial Access

1. You'll receive an email to your personal address with login credentials
2. Go to [mail.google.com](https://mail.google.com)
3. Enter your @intime.com email
4. Use the temporary password provided
5. Set a new strong password

### Security Setup

- [ ] Enable 2-Factor Authentication
  - Go to Security settings
  - Set up 2FA with phone or authenticator app
- [ ] Add recovery email (personal email)
- [ ] Add recovery phone number

### Email Signature

Set up your signature:

```
[Your Full Name]
[Your Title]
InTime Staffing

Phone: [Your Number]
Email: [your.email@intime.com]
Web: www.intime.com
```

### Key Settings

- [ ] Set timezone correctly
- [ ] Configure vacation responder (when needed)
- [ ] Set up email filters for important messages
- [ ] Add team members to contacts

## Security Requirements

### Password Policy

- Minimum 12 characters
- Mix of upper, lower, numbers, symbols
- Don't reuse passwords
- Change every 90 days
- Use password manager (recommended)

### 2-Factor Authentication

Required for:
- Email
- InTime platform
- Any system with sensitive data

### Data Security

- Don't share login credentials
- Lock computer when away
- Don't access from public WiFi without VPN
- Report suspicious activity immediately

## Troubleshooting

### Can't Login

1. Check caps lock
2. Try password reset
3. Clear browser cache
4. Try different browser
5. Contact IT support

### Tool Not Working

1. Refresh the page
2. Clear cache/cookies
3. Try different browser
4. Check internet connection
5. Contact IT support

### Getting Help

| Issue | Contact |
|-------|---------|
| Login problems | IT Support (Slack #tech-support) |
