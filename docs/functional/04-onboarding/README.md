## Quick Start

### Day 1 Checklist
- [ ] Complete HR paperwork
- [ ] Get system access (email, Slack, InTime)
- [ ] Read [Company Bible](../01-foundation/COMPANY_BIBLE.md)
- [ ] Meet your team lead
- [ ] Set up your workspace

### Week 1 Goals
- [ ] Complete role-specific handbook
- [ ] Shadow team members
- [ ] Learn the business model
- [ ] Attend all team meetings
- [ ] Complete US IT Staffing 101

### Month 1 Goals
- [ ] Handle first tasks independently
- [ ] Know all team processes
- [ ] Build relationships across team
- [ ] Complete 30-day check-in
