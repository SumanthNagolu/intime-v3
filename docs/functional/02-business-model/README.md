## Quick Reference

### Triple Monetization
```
Training Fee (₹1.5L) → FTE Commitment Fee (₹25K) → Placement Margin (15-25%)
```

### Seven Funnels
1. **Academy** - Train new IT professionals (₹1.5L/student)
2. **Recruiting** - Source experienced talent
3. **Campaigns** - Attract inbound candidates
4. **Account Acquisition** - Win new clients
5. **Job Fulfillment** - Fill client positions
6. **Bench Sales** - Place available W2 consultants
7. **Delivery** - Ensure engagement success

### Phase 1 Team (16 people)
- **Supply Side** (8): Training (4) + Screening (4)
- **Demand Side** (7): BDM (1) + Bench (3) + Recruiting (3)
- **Delivery** (1): Delivery Manager

### Phase 1 Economics
- Monthly burn: ₹12L
- Break-even: 7 students enrolled OR 3-4 placements/month
