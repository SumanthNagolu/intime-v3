## THE SEVEN FUNNELS

### Supply Side Funnels (Where talent comes from)

| # | Funnel | Purpose | Output |
|---|--------|---------|--------|
| 1 | **Academy** | Train new IT talent from scratch | Graduates → Pool or Bench |
| 2 | **Recruiting** | Source experienced professionals | Qualified candidates → Pool |
| 3 | **Sales Campaigns** | Attract inbound candidates | Respondents → Pool |

### Demand Side Funnels (Where talent goes)

| # | Funnel | Purpose | Output |
|---|--------|---------|--------|
| 4 | **Account Acquisition** | Get new client accounts | Active clients with jobs |
| 5 | **Job Fulfillment** | Fill client positions | Placements |
| 6 | **Bench Sales** | Sell available consultants | Placements |
| 7 | **Delivery** | Ensure engagement success | Renewals, referrals |

## FUNNEL 1: ACADEMY (Training Pipeline)

### Purpose
Train people for IT careers. Graduates become your talent supply.

### Who Enrolls
- Career changers
- Recent graduates (non-IT degrees)
- People wanting to enter tech
- Upskilling from adjacent fields

### The Flow

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  ATTRACT    │────▶│   ENROLL    │────▶│   TRAIN     │────▶│  GRADUATE   │────▶│   PLACE     │
│             │     │             │     │             │     │             │     │             │
│ Marketing   │     │ Application │     │ Coursework  │     │ Assessment  │     │ To pool     │
│ Outreach    │     │ Selection   │     │ Projects    │     │ Certification│    │ To bench    │
│ Referrals   │     │ Onboarding  │     │ Mentorship  │     │ Portfolio   │     │ Direct hire │
│ Partnerships│     │             │     │             │     │             │     │             │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
```

### Who Owns It
**Academy Team:**
- Academy Manager (runs the program)
- Instructors (teach courses)
- Academy Coordinator (admin, student support)
- Career Coach (placement support)

### Inputs
- Students (via marketing, partnerships, referrals)
- Curriculum (courses, projects, assessments)
- Instructors/mentors

### Outputs
- Trained IT professionals
- Certified in specific technologies
- Portfolio of projects
- Ready for entry-level to mid-level roles

### Key Metrics
| Metric | Target |
|--------|--------|
| Enrollment rate | Based on capacity |
| Completion rate | 85%+ |
| Certification pass rate | 90%+ |
| Time to first placement | < 30 days post-graduation |
| Graduate placement rate | 90%+ within 90 days |
| Graduate retention (at placement) | 85%+ at 6 months |

### Campaigns That Feed This Funnel
1. **Student acquisition marketing** - Social, content, ads
2. **Partnership programs** - Universities, community colleges
3. **Corporate sponsorship** - Companies sponsor students
4. **Referral programs** - Graduate referrals
5. **Career fair presence** - Attract career changers

## FUNNEL 2: RECRUITING (Experienced Talent Pool Building)

### Purpose
Source and qualify experienced IT professionals for the talent pool.

### The Flow

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   SOURCE    │────▶│   ENGAGE    │────▶│   SCREEN    │────▶│  QUALIFY    │────▶│    POOL     │
│             │     │             │     │             │     │             │     │             │
│ Job boards  │     │ Outreach    │     │ Phone       │     │ Deep call   │     │ Tagged      │
│ LinkedIn    │     │ Response    │     │ screening   │     │ Work history│     │ Searchable  │
│ Referrals   │     │ handling    │     │ Basic fit   │     │ Projects    │     │ Fresh       │
│ Inbound     │     │             │     │             │     │ Own words   │     │ Ready       │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
```

### Who Owns It
**Back Office Recruiter (Pool Builder)**

### Key Differentiator: Deep Qualification
- Not just resume review
- Detailed work history captured
- Project descriptions in candidate's own words
- Technical depth understood
- Availability and expectations known

### Inputs
- Dummy requirements (skill profiles to build)
- Account Manager intel (what accounts need)
- Market demand signals

### Outputs
- Experienced professionals with rich profiles
- Ready to match when jobs come

### Key Metrics
| Metric | Target |
|--------|--------|
| Candidates sourced/week | 50+ |
| Engagement rate | 30%+ |
| Screen pass rate | 50%+ |
| Qualification rate | 60%+ |
| Pool freshness (< 30 days) | 80%+ |

### Campaigns That Feed This Funnel
1. **Job board postings** - Always on for key skills
2. **LinkedIn sourcing** - Targeted outreach
3. **Referral programs** - From placements and pool
4. **Skill-specific blitzes** - Fill gaps fast
5. **Re-engagement campaigns** - Warm up cold candidates

## FUNNEL 5: JOB FULFILLMENT

### Purpose
Fill client jobs quickly with quality talent from the pool.

### The Flow

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  JOB IN     │────▶│   MATCH     │────▶│   SUBMIT    │────▶│ INTERVIEW   │────▶│   PLACE     │
│             │     │             │     │             │     │             │     │             │
│ From client │     │ From pool   │     │ Confirm     │     │ Coordinate  │     │ Offer       │
│ Requirements│     │ From bench  │     │ availability│     │ Feedback    │     │ Accept      │
│             │     │ From academy│     │ Present     │     │ Advance     │     │ Start       │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
```

### Candidate Sources (in priority order)
1. **Bench** - Already your employees, available now
2. **Pool** - Pre-qualified, ready to go
3. **Academy graduates** - Fresh talent, trained by you
4. **New sourcing** - If pool doesn't have fit

### Who Owns It
- **Job Creation:** Account Manager
- **Matching:** System + Front Office
- **Filling:** Front Office Recruiter

### Key Metrics
| Metric | Target |
|--------|--------|
| Time to first submit | < 24 hours |
| Submissions per job | 3-5 |
| Interview rate | 50%+ |
| Offer rate | 30%+ |
| Fill rate | 70%+ |
| Time to fill | < 14 days |

## SUMMARY

### Three Talent Sources
1. **Academy** - Train new talent (students → graduates)
2. **Recruiting** - Source experienced talent
3. **Campaigns** - Attract inbound talent

### One Talent Pool
- All sources feed into unified pool
- Rich profiles, pre-qualified
- Ready to match and deploy

### Three Deployment Paths
1. **Bench** - Hire to your payroll, sell proactively
2. **Contract** - Place at clients on contract
3. **Direct Hire** - Place in FTE roles

### Client Side
- Business Dev wins accounts
- Account Managers own relationships and jobs
- Front Office fills jobs
- Delivery ensures success

### The Flywheel
```
Academy graduates → Pool/Bench → Placements → Success →
Renewals → Reputation → More students → More clients → Growth
```
