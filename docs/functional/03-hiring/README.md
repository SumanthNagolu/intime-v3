## Quick Reference

### Phase 1 Hiring Strategy
**Senior + Intern Model**
- 7 Seniors (₹75K-₹3L/month) - Pod leads who train
- 9 Interns (₹15K-₹25K/month) - Learn and grow

### Hiring Phases
| Phase | Timeline | Hires |
|-------|----------|-------|
| Phase 1 | Week 1-2 | 4 core seniors (Trainer, Training Lead, Bench Lead, BDM) |
| Phase 2 | Week 3-4 | 2 seniors + 2 interns |
| Phase 3 | Month 2 | Delivery Manager + 7 remaining interns |

### India Compensation Bands
| Level | Monthly Range |
|-------|---------------|
| Intern | ₹12K - ₹25K |
| Associate | ₹25K - ₹50K |
| Senior | ₹50K - ₹1L |
| Lead | ₹1L - ₹2L |
| Manager | ₹2L+ |

### Key Roles to Hire First
1. **Guidewire Trainer** (₹3L) - Core product delivery
2. **Training Lead** (₹75K) - Sales and coordination
3. **Bench Lead** (₹1L) - Revenue generation
4. **BDM** (₹1.5L) - Client acquisition
