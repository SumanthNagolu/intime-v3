## CANDIDATE MATCHING SCORING RUBRIC

**Instructions**: Use this rubric to score candidates on their fit for this role. Each dimension should be scored 1-5.

| Scoring Dimension | Weight | Scoring Guide | Score |
|---|---|---|---|
| **Must-Have Technical Skills** | [%] | 5: All expert, recent / 4: All present, some intermediate / 3: Most present / 2: Some missing / 1: Major gaps | [__] |
| **Years of Experience** | [%] | 5: Exceeds target / 4: Meets target / 3: Slightly below / 2: Significantly below / 1: Way below | [__] |
| **Domain/Industry Knowledge** | [%] | 5: Deep expertise in this domain / 4: Solid background / 3: Some exposure / 2: Minimal / 1: None | [__] |
| **Seniority/Maturity Fit** | [%] | 5: Perfect level / 4: Slightly overqualified / 3: Right level / 2: Slightly junior / 1: Way too junior | [__] |
| **Soft Skills & Communication** | [%] | 5: Excellent / 4: Strong / 3: Adequate / 2: Concerning / 1: Red flag | [__] |
| **Remote Work Capability** | [%] | 5: Ideal / 4: Strong / 3: Capable / 2: Requires adjustment / 1: Not suitable | [__] |
| **Availability & Timeline** | [%] | 5: Available immediately / 4: <1 week notice / 3: 1-2 weeks / 2: 2-4 weeks / 1: >4 weeks | [__] |
| **Rate Fit** | [%] | 5: Below budget / 4: At budget / 3: Slightly above / 2: Significantly above / 1: Way above budget | [__] |
| **Overall Interview Performance** | [%] | 5: Outstanding / 4: Strong / 3: Good / 2: Acceptable / 1: Not suitable | [__] |
| **Culture & Team Fit** | [%] | 5: Excellent fit / 4: Strong fit / 3: Adequate / 2: Some concerns / 1: Poor fit | [__] |

**Scoring Calculation**: Multiply each score by its weight, sum all weighted scores to get **Total Candidate Score: [__]/5.0**

**Candidate Recommendation Guidelines**:
- **4.5+**: Strong YES - Move forward aggressively
- **3.5-4.4**: YES - Good fit, move forward
- **2.5-3.4**: MAYBE - Explore further, interview recommended
- **1.5-2.4**: WEAK - Consider only if no other options
- **Below 1.5**: NO - Do not submit
