### Compensation Structure (India)

| Level | Monthly Range (INR) |
|-------|---------------------|
| Intern | ₹12,000 - ₹25,000 |
| Associate | ₹25,000 - ₹50,000 |
| Senior | ₹50,000 - ₹1,00,000 |
| Lead | ₹1,00,000 - ₹2,00,000 |
| Manager | ₹2,00,000+ |

| Cadence | Purpose |
|---------|---------|

| Monthly | Performance conversation |
| Quarterly | OKR review and goal setting |
| Annual | Comprehensive review, compensation |

2. **Integrity** - Be honest in all dealings
3. **Professionalism** - Represent InTime well
4. **Excellence** - Do your best work
