## Part 3: Timesheet Management

### The Timesheet Cycle

```
Week Start (Sunday)
        │
        ▼
Contractor Works & Logs Hours (Mon-Fri)
        │
        ▼
Timesheet Submission (by Monday noon following week)
        │
        ▼
You Verify (Monday afternoon)
        │
        ▼
Client/Manager Approves (by Tuesday)
        │
        ▼
Payroll Processing (Wednesday)
        │
        ▼
Payment (Friday)
```

### Timesheet Collection Process

**Monday Morning:**
1. Check who submitted timesheets
2. Identify missing timesheets
3. Send reminders to missing contractors

**Reminder Email (Friendly):**
```
Subject: Timesheet Reminder - Week of [Date]

Hi [Name],

Just a reminder that timesheets are due by noon today for
the week of [date range].

Please submit through [system/method] as soon as possible.

Let me know if you have any questions!

Thanks,
[Your name]
```

**Reminder Email (Urgent):**
```
Subject: URGENT: Timesheet Overdue - Week of [Date]

Hi [Name],

Your timesheet for week of [date] is overdue. Please submit
immediately to ensure on-time payment.

If you're having issues submitting, please call me at [phone].

Thanks,
[Your name]
```

### Timesheet Verification

**Check for each timesheet:**
```
□ Hours add up correctly
□ Hours match expected (40/wk standard)
□ Overtime approved (if any)
□ Dates are correct week
□ Project codes correct
□ No duplicate entries
□ Contractor signature/approval
```

**Common Errors:**
| Error | Action |
|-------|--------|
| Wrong dates | Return to contractor to fix |
| Math error | Fix if minor, return if major |
| Overtime without approval | Verify with manager before processing |
| Missing project code | Check with contractor or manager |
| Duplicate submission | Keep only one, verify correct week |

### Handling Timesheet Issues

**Contractor Didn't Submit:**
```
Day 1: Send reminder
Day 2: Call contractor, escalate to recruiter
Day 3: Final warning - may miss payroll
Day 4+: Escalate to Team Lead
```

**Client Didn't Approve:**
```
Day 1: Send reminder to client
Day 2: Escalate to Account Manager
Day 3: Process with internal approval, document
```

## Part 4: Payroll Processing

### Payroll Timeline (Bi-Weekly)

| Day | Activity |
|-----|----------|
| Monday | Collect and verify all timesheets |
| Tuesday | Finalize hours, get approvals |
| Wednesday | Submit to payroll, generate reports |
| Thursday | Payroll processes, verify |
| Friday | Payments sent/received |

### Payroll Checklist

**Before Submitting:**
```
□ All timesheets verified
□ All hours approved
□ New hire info complete
□ Changes processed (address, banking, etc.)
□ Deductions calculated
□ Benefits applied (if applicable)
```

**Payroll Report Format:**
```
PAYROLL REPORT - Pay Period: [Date Range]

Contractor Name | Hours | Rate | Gross | Deductions | Net
---------------|-------|------|-------|------------|----
[Name]         | 40    | $60  | $2400 | $400       | $2000
[Name]         | 40    | $55  | $2200 | $350       | $1850

TOTALS         | 80    |      | $4600 | $750       | $3850
```

### Handling Payroll Issues

**Late Timesheet:**
- If caught before cutoff: Include in payroll
- If after cutoff: Process in next payroll, notify contractor

**Pay Rate Discrepancy:**
1. Verify correct rate in system
2. Check offer letter/contract
3. If error, calculate correction
4. Process adjustment in next payroll
5. Communicate with contractor

**Banking Error:**
1. Verify banking details
2. Reprocess payment
3. Notify contractor of delay
4. Consider alternative (check, wire)

## Part 6: Compliance & Documentation

### New Placement Onboarding

**Required Documents:**
```
□ Signed offer letter/contract
□ I-9 form (completed)
□ W-4 form (tax withholding)
□ Direct deposit form
□ ID copies (for I-9)
□ Background check authorization
□ Drug test consent (if required)
□ NDA/Confidentiality agreement
□ Client-specific forms
```

### Background Check Process

1. **Send Authorization**
   - Collect signed consent form
   - Gather required information

2. **Initiate Check**
   - Submit to background check provider
   - Track status

3. **Review Results**
   - Clean: Proceed with placement
   - Issues: Escalate to Team Lead

4. **Document**
   - Save results securely
   - Update system status

**Timeline Expectations:**
| Check Type | Typical Time |
|------------|--------------|
| Criminal | 1-3 days |
| Education | 3-5 days |
| Employment | 3-7 days |
| Complete Package | 5-10 days |

### I-9 Verification

**Section 1:** Completed by employee (Day 1)
**Section 2:** Completed by employer (within 3 days)

**Acceptable Documents:**
| List A (Identity + Work) | List B (Identity) | List C (Work) |
|--------------------------|-------------------|---------------|
| US Passport | Driver's License | Social Security Card |
| Passport Card | State ID | Birth Certificate |
| Permanent Resident Card | School ID | Employment Authorization |
| Employment Authorization | Voter Registration | |

### Document Storage

**Organization:**
```
/Contractors
  /[Contractor Name]
    /Offer & Contract
    /Tax Documents (W-4, W-9)
    /ID Copies
    /Background Check
    /I-9
    /Timesheets
    /Correspondence
```

**Retention:**
| Document Type | Retention Period |
|---------------|------------------|
| I-9 | 3 years after hire or 1 year after termination |
| Tax documents | 7 years |
| Timesheets | 3 years |
| Contracts | Duration + 3 years |

## Part 10: Common Scenarios

### Scenario: Contractor Paid Wrong Amount

**Steps:**
1. Verify the error (check calculations)
2. Determine cause (hours, rate, deduction)
3. Calculate correct amount
4. Communicate with contractor
5. Process correction
6. Document the issue

**Communication:**
```
"[Name], I apologize for the pay discrepancy. After reviewing,
I found [explanation of error].

The correct amount should have been $[X]. I'm processing a
correction of $[difference] which will be in your [next pay /
separate payment] on [date].

I'm sorry for the inconvenience. Please let me know if you
have any questions."
```

### Scenario: Client Disputes Invoice

**Steps:**
1. Listen to their concern
2. Review the invoice and timesheets
3. Compare with records
4. If error: Correct and resend
5. If correct: Provide supporting documentation
6. Escalate if unresolved

**Communication:**
```
"Thank you for bringing this to our attention. Let me review
the invoice against our timesheet records.

[After review]

I've verified the invoice against the approved timesheets
for [period]. The hours and rate are correct as shown.

I've attached the timesheet copies for your reference.

Please let me know if you need anything else to process
payment."
```

### Scenario: Missing Documentation

**Steps:**
1. Identify what's missing
2. Contact contractor
3. Set deadline
4. Follow up
5. Escalate if not received

**Communication:**
```
"Hi [Name], I'm completing your file and noticed we're missing:
- [Document 1]
- [Document 2]

Please submit these by [date] to ensure everything is in order.

If you have questions about any of these documents, let me know.

Thanks,
[Your name]"
```
