# Role Handbooks

Complete operational guides for every role in the recruiting team.

---

## Handbook Structure

Each role handbook contains:

| Section | Contents |
|---------|----------|
| **Role Overview** | Mission, responsibilities, success metrics |
| **Day in the Life** | Hour-by-hour breakdown of typical day |
| **Key Processes** | Step-by-step workflows for key activities |
| **Scripts & Templates** | Call scripts, email templates |
| **Common Scenarios** | How to handle typical situations |
| **Tools & Systems** | How to use each tool effectively |
| **Metrics & Reporting** | What to track and how |
| **Career Path** | Growth opportunities and promotion criteria |

## How to Use These Handbooks

### For New Hires

1. Read your role handbook completely in Week 1
2. Reference scripts and templates daily
3. Use checklists for every process
4. Ask questions when unclear

### For Managers

1. Use handbooks for training new team members
2. Reference for performance coaching
3. Update based on what works
4. Ensure consistency across team

### For Continuous Improvement

- Handbooks are living documents
- Suggest improvements anytime
- Best practices get added
- Outdated content gets removed
