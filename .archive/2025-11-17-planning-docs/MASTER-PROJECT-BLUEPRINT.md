### Database Architecture

**Standard Schema Pattern** (all tables):
```typescript
{
  id: uuid,                    // Primary key
  // Business fields here
  createdAt: timestamp,        // Audit trail
  updatedAt: timestamp,        // Audit trail
  createdBy: uuid,             // Who created
  updatedBy: uuid,             // Who updated
  deletedAt: timestamp | null, // Soft delete
  orgId: uuid,                 // Multi-tenancy
}
```

**Security:**
- Row Level Security (RLS) on ALL tables
- Users can ONLY see their org's data
- No way to bypass (enforced at database level)
